# 🐵 Koji Hunt: Multi-Language Hardcore Edition

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-2.2.0-gold.svg)
![Platform](https://img.shields.io/badge/platform-Web-brightgreen.svg)

**Koji Hunt** adalah mini-game web bertema cyberpunk yang menantang fokus dan kecepatan tangan. Pemain mengontrol **Koji** menggunakan D-Pad untuk menangkap pisang dan menghindari bom dalam lingkungan yang tingkat kesulitannya meningkat secara dinamis.

## ✨ Fitur Utama
- **Multi-Language:** Indonesia, English, & Japanese (日本語).
- **Responsive D-Pad:** Kontrol tombol yang halus dan tanpa jeda (zero-delay).
- **Hardcore Progression:** Kecepatan jatuh dan frekuensi item meningkat setiap 5 detik.
- **Rank System:** Dapatkan gelar "Web3 Legend" jika skor kamu menembus 1000 poin.

## 🎮 Cara Bermain
1. Pilih bahasa di menu awal.
2. Gunakan tombol **◀** dan **▶** di layar untuk bergerak.
3. Ambil pisang (+20) dan hindari bom (-70).
4. Bertahanlah hingga waktu habis!
